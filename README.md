# Sakura PS4 GamepadViewer Skin v4 — Concept Image Version

This version uses your provided concept controller image as the controller art.

Upload these files to your GitHub Pages repo root:

- `edit.css`
- `controller-concept.png`

Use this GamepadViewer URL:

`https://gamepadviewer.com/?p=1&s=5&editcss=https%3A%2F%2Fdp12-34.github.io%2Fps4-controllers-game-pad-viewer%2Fedit.css`

After uploading:

1. Commit the files.
2. Wait 1–2 minutes.
3. Hard refresh GamepadViewer with Ctrl + Shift + R.
4. Put the full GamepadViewer URL into OBS.

Note:
This is the closest route to the exact image style. Since the image already has the buttons and sticks drawn, the default GamepadViewer art is hidden. Pressed inputs still show as a subtle glow.
